# AIRoR

Artificial intelligence for Rigs of Rods.

Installation:

sudo apt install python3-libevdev

copy AIRoR_fake_joystick_device.map and input.map in ~/.rigsofrods/config

Launch:

AIROR.py <ROR binary> <Configuration file *.json>
